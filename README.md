# open-recruitment.github.io
- 本仓库说明（若有需求，可Fork后提交，无需求，则闲置）
- 这里是由Rain提供的"全球公开招聘信息库"
- 声明1：本人不参与实际管理，也不参与审核，也不提供任何提交权限，只提供fork，各种招聘方，管理各自的招聘信息。
- 声明2：应聘与招聘，由双方自行达成意愿，均由双方自行负责。
- 声明3：本人不收受双方任何一方，任何费用，各自达成意愿后，因各种因素造成的损失与本人无关。
- 声明4：招聘信息，如果不实，涉嫌虚假内容或诈骗信息，可提交issues举报，将会设置招聘黑名单展露。（相同的虚假内容，可在同一个issues中引用举报）
- 声明5：如果使用本仓库信息，建议关注本仓库，可以及时获得仓库的信息变更及 issues 通知。
- 声明6：招聘信息由各个招聘方自行提交，请务必真实有效。
- 声明7：不得涉及各国政治、暴力、色情、以及各种非法言论，将会设置招聘黑名单展露。
- 声明8：请一切内容，在合法合规前提下进行，造就一个简洁、绿色、纯粹的招聘信息库。
- 
- 特别注意：应聘人员，应本能的不信任本仓库提供的任何招聘信息，以此为前提对招聘方信息进行严格核对，避免上当受骗。
- 由于网络上的诈骗信息猖獗，且手段层出不穷，任何招聘信息都应该以不信任的态度去核对。
- 比如 web3 去中心化的核心信任机制，就是绝对不信任的情况下存生出来的。
- 本仓库存的目的，是考虑到有人需要这样的招聘与应聘入口，简单-直接-高效，而且应该可以过滤掉大多数垃圾无效信息。
- 但是，请不要滥用本仓库。
- 
- **用户可通过 Fork + Pull Request 在各自的子仓库中提交新招聘信息**
- **通过本仓库，可链接到所有子仓库，方便招聘者发布信息，也方便应聘者寻找信息**
- **所有虚假信息，虚假招聘、非法言论的信息，可在本主库中提交issues，根据情况设置黑名单披露**
